== Description ==

Adds features to your Wordpress site designed by [Wagga Wagga Web](http://www.waggawaggaweb.com.au/) for ease of use including shortcodes and other fixes.

== Installation ==

1. Upload folder `wagga-wagga-web-custom-features` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
